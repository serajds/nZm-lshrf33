أريد بناء نظام ويب احترافي متكامل لإدارة ومتابعة أعمال الإشراف الهندسي على مشاريع البناء، باستخدام Python (Flask أو FastAPI) مع واجهة مستخدم حديثة باللغة العربية بالكامل وتصميم احترافي موحد.
🎯 الهدف من النظام:
إنشاء منصة مركزية لإدارة مشاريع البناء من ناحية الإشراف الهندسي، تتضمن تتبع التقدم، إدارة التقارير، مراقبة الجدول الزمني، وتحليل الانحرافات، مع إمكانية مشاركة البيانات مع الجهة المالكة عبر رابط خاص.
________________________________________
🧩 المميزات الأساسية:
1. إدارة المشاريع:
•	إنشاء مشروع جديد يحتوي على:
o	اسم المشروع
o	الموقع
o	الجهة المالكة
o	الجهة المشرفة
o	المقاول
o	تاريخ البدء والانتهاء المتوقع
•	إمكانية تعديل وحذف المشاريع
________________________________________
2. إدارة الجدول الزمني:
•	إدخال بنود المشروع (Activities)
•	لكل بند:
o	اسم البند
o	تاريخ البداية
o	تاريخ النهاية
o	نسبة الإنجاز
•	عرض الجدول الزمني على شكل:
o	Gantt Chart
•	مقارنة المخطط مع الفعلي (Planned vs Actual)
________________________________________
3. التقارير الدورية:
•	إنشاء تقارير:
o	أسبوعية
o	شهرية
•	كل تقرير يحتوي على:
o	وصف الأعمال المنفذة
o	نسبة الإنجاز الحالية
o	ملاحظات فنية
o	توصيات
o	صور مرفقة
•	حفظ التقارير وربطها بالمشروع
________________________________________
4. تحليل الأداء والانحراف:
•	حساب الانحراف الزمني:
o	تأخير أو تقدم
•	حساب الانحراف في نسبة الإنجاز
•	عرض مؤشرات:
o	Progress %
o	Delay Days
•	عرض Dashboard يحتوي على:
o	حالة المشروع
o	الرسوم البيانية
________________________________________
5. إدارة الملفات:
•	رفع ملفات المشروع:
o	صور
o	تقارير PDF
o	نتائج اختبارات
o	مستندات
•	تنظيم الملفات حسب:
o	نوع الملف
o	تاريخ الرفع
•	ربط النظام مع OneDrive API لتخزين الملفات
________________________________________
6. تصدير التقارير:
•	إنشاء تقارير PDF احترافية تلقائياً
•	تنسيق التقرير يشمل:
o	شعار المشروع
o	بيانات المشروع
o	ملخص تنفيذي
o	الجداول الزمنية
o	الصور والملاحظات
•	إمكانية تحميل التقرير أو مشاركته
________________________________________
7. لوحة تحكم للجهة المالكة:
•	إنشاء رابط خاص لكل مشروع
•	الدخول بكلمة مرور فقط
•	عرض:
o	تقدم المشروع
o	التقارير
o	الرسوم البيانية
•	بدون صلاحيات تعديل (عرض فقط)
________________________________________
8. نظام المستخدمين والصلاحيات:
•	تسجيل دخول
•	أدوار المستخدمين:
o	مدير
o	مهندس مشرف
o	جهة مالكة (عرض فقط)
•	التحكم في الصلاحيات
________________________________________
🎨 متطلبات الواجهة:
•	اللغة: العربية بالكامل
•	اتجاه: RTL
•	تصميم حديث (Dashboard احترافي)
•	استخدام:
o	Charts (مثل Chart.js)
o	جداول تفاعلية
________________________________________
⚙️ التقنيات المقترحة:
•	Backend: Python (Flask أو FastAPI)
•	Frontend: HTML + CSS + JavaScript (أو React)
•	Database: PostgreSQL أو SQLite
•	تخزين الملفات: OneDrive API
•	إنشاء PDF: مكتبة مثل ReportLab أو WeasyPrint

•	دعم تعدد المشاريع
